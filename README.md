## jay-vscode

> 杰伦提醒休息喝奶茶时间到了！！！！

就是这样的
也可以直接 F1 输入 callJay 召唤
也可以右键菜单召唤

为杰伦打 call！！！！

```json
  // 让杰伦提醒的间隔时间（分钟）
  "jay.timer": 120,
  // 让杰伦提醒的文字
  "jay.text": "哎哟，不错哦，这代码，不过记得先休息一下，喝杯奶茶吧~",
  // 杰伦的帅照!!!  可供选择：["jay-1","jay-2"]  也可以是网络图片地址
  "jay.picture": "jay-2",
```
